# MindCare — Kesehatan Mental Mahasiswa

Website ini dibuat sebagai project edukatif untuk memberikan informasi mengenai kesehatan mental mahasiswa, tekanan akademik, burnout, serta langkah-langkah sederhana yang dapat dilakukan untuk menjaga kondisi diri selama menjalani perkuliahan.

## Deskripsi Project

Website ini berisi informasi tentang:

- Permasalahan kesehatan mental yang sering dialami mahasiswa
- Tekanan akademik, tekanan sosial, overthinking, dan pola istirahat yang terganggu
- Tanda-tanda burnout secara fisik, emosional, dan akademik
- Solusi edukatif untuk mengelola waktu, istirahat, overthinking, dan dukungan sosial
- Panduan kapan mahasiswa perlu mencari bantuan dari lingkungan terdekat atau tenaga profesional
- Pesan penutup untuk mengingatkan pentingnya menjaga kesehatan diri

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript (jika ditambahkan pada tahap berikutnya)

## Tujuan

Meningkatkan kesadaran mahasiswa terhadap pentingnya menjaga kesehatan mental, mengenali tanda-tanda tekanan akademik dan burnout, serta mendorong mahasiswa untuk berani mencari bantuan ketika kondisi mental mulai terasa berat.

## Cara Menjalankan

1. Clone atau download repository
2. Pastikan file `index.html` berada di dalam folder project
3. Buka file `index.html` di browser
4. Website dapat dijalankan secara offline tanpa instalasi tambahan

## Catatan

Website ini bersifat edukatif dan bukan pengganti diagnosis, konseling, terapi, atau bantuan profesional dari tenaga kesehatan mental. Jika mengalami tekanan mental yang berat, segera hubungi orang terdekat, layanan kesehatan, atau tenaga profesional.
